Shady Meadows: Full-Stack Automation Framework
This repository contains a professional-grade, full-stack automation framework built to ensure the quality and reliability of the Shady Meadows Hotel Management Platform. By combining UI automation and API contract testing
Smoke Suite: A high-priority suite designed for rapid feedback on core business functions (Login, Room Availability).
Tech Stack
Language: Python 3.11
Automation: Playwright (Python)
Test Runner: Pytest
Reporting: Pytest-HTML (with self-contained assets)
The framework is integrated with GitHub Actions to implement a Continuous Integration (CI) pipeline
